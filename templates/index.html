1. Overview

EPC Structural Design Software is a web-based pipe-rack and equipment support design tool that follows the AISC 360-22 (steel) and ACI 318-19 (concrete) codes with seismic and wind loading per ASCE 7-22. It is analogous in scope to AVEVA E3D or Intergraph S3D but runs locally with no licence server.

  • Backend — Python / Flask + SQLite (WAL mode)
  • Frontend — Single-page HTML5 with Bootstrap 5 and Three.js WebGL
  • Units — Internal: N and mm. User input / reports: kN, kN·m, mm

2. Recommended Workflow

  1. Create / open a Project and set wind, seismic and soil parameters.
  2. Import or add Equipment (tanks, vessels, reactors, pumps …).
  3. Import Nozzle loads from the piping stress package (Fx/Fy/Fz/Mx/My/Mz).
  4. Define Structure — rack geometry (bays, tiers, widths) then click Generate Grid to auto-create nodes, members, and equipment support framing.
  5. Review / add Load cases and combinations.
  6. Open Calculate tab and run Section Checks, Reactions and Foundation Design.
  7. Review Results tables and export CSV.
  8. Inspect the model in the 3D View.

4. Equipment

Equipment items carry operating weight, empty weight, test weight, centre of gravity, and physical dimensions used for both structural load calculation and 3D visualisation.

Adding equipment manually

Click Add Equipment on the Equipment tab. Required fields: Tag, Type, Operating Weight (kg). Provide Tier and Bay to locate the equipment on the rack; the 3D view then positions it automatically based on rack geometry.

CSV import

Download equipment_sample.csv (Import/Export tab) as a template. Key columns:

5. Nozzle Loads

Nozzle loads from the piping stress analysis (Caesar II, AutoPIPE, etc.) are entered per nozzle and are automatically summed and applied to the rack node at the equipment's tier / bay location under the OL (Operating Load) load case.

Import nozzle loads from CSV
  1. On the Equipment tab click Import Nozzle Loads CSV, or go to Import/Export and select Nozzle Loads.
  2. Ensure all EQUIPMENT_TAG values match existing equipment in the project.
  3. Upload the file and click Import. The tool reports how many nodal loads were generated.

6. Load Cases

Ten standard load cases are seeded automatically for each project:

CaseDescription
DLDead load — self-weight of structure + equipment empty weight
LLLive load — piping contents, maintenance loads (typically 4.8 kPa pipe deck)
WL_X / WL_ZWind loads in X and Z directions (ASCE 7-22 simplified open frame)
SL_X / SL_ZSeismic ELF forces in X and Z directions
TLThermal expansion loads
OLOperating load — nozzle loads aggregated per rack node
TPTest pressure load — hydro-test weight
FIFriction/anchor loads at pipe supports

Load combinations following ASCE 7-22 LRFD (§2.3) and ASD (§2.4) are auto-generated via Calculate → Generate Combinations.

Generate Grid

Click Generate Grid after defining geometry. The software creates:

  • Main rack — columns (A/B lines), transverse beams, longitudinal stringers, X-braces
  • Equipment supports — saddle beams for horizontal vessels/exchangers, skid frames for pumps/compressors, leg connections for vertical vessels/reactors

All members are assigned default sections (lightest W-section from the database) and A992 material. Use the Structure tab to change individual section assignments before running checks.

9. Results

The Results tab shows three tables:

  • Member Results — Unity Check (UC) per member per load combination. UC > 1.0 = FAIL (shown in red).
  • Support Reactions — Axial (Pz), shear (Vx, Vz), and moment (Mx, Mz) at each column base per load case.
  • Foundation — Footing dimensions, rebar schedule, and pass/fail for each ACI check.

Click Export Results CSV to download all member checks as a spreadsheet.

10. Import / Export

All imports require a project to be selected. Supported import types:

  • Equipment CSV — bulk-load equipment list (download sample for format)
  • Nozzle Loads CSV — nozzle forces/moments from piping stress tool

Export: Member Results CSV contains tag, section, material, governing combo, Pu/φPn, Mu/φMn, shear UC, axial UC, combined UC and pass/fail status for every member.
